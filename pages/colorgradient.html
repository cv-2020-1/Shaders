COLOR GRADIENT

DESCRIPCIÓN DEL COLOR GRADIENT

El fondo es un degradado de color y progresa de gris oscuro a gris claro. La barra horizontal parece progresar de gris claro a gris oscuro, pero en realidad es de un solo color. Si mantiene presionado el click izquierdo, cambiará el color del fondo pero no el color de la barra horizontal.

¿COMO SE USA?

Quedate observando el rectangulo del medio y veras que pareciera con degradado pero luego Haz click sobre la pantalla y veras que no tiene degrade

CODIGO FRAGMENT SHADER

#ifdef GL_ES
precision mediump float;
#endif

uniform vec2 u_resolution;
uniform bool u_mouse;

void main() {
  vec2 st = gl_FragCoord.xy/u_resolution.xy;
  st.x = 0.8*st.x + 0.1;

  if(u_mouse){
    gl_FragColor = vec4(st.x,st.x,st.x,1.0); // R,G,B,A
  }else{
    gl_FragColor = vec4(0.31,0.31,0.31,1.0); // R,G,B,A
  }
}

CODIGO VERTEX SHADER

#ifdef GL_ES
precision mediump float;
#endif

attribute vec3 aPosition;

void main() {
  vec4 positionVec4 = vec4(aPosition, 1);
  positionVec4.xy = positionVec4.xy * 2.0 - 1.0;
  gl_Position = positionVec4;
}

CODIGO VERTEX SHADER 2

#ifdef GL_ES
precision mediump float;
#endif

attribute vec3 aPosition;

void main() {
  vec4 positionVec4 = vec4(aPosition, 1);

  positionVec4.y = positionVec4.y*0.5;
  positionVec4.xy = positionVec4.xy - 0.5;
  positionVec4.y = positionVec4.y + 0.25;

  gl_Position = positionVec4;
}
